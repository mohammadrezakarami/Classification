# Classification

This repository collects multiple supervised classification notebook implementations using a shared teaching dataset and standard machine learning models.

## Repository Contents

- `logistic_regression.ipynb`
- `KNN.ipynb`
- `SVM.ipynb`
- `decision_tree_classifier.ipynb`
- `Random_forest_classifier.ipynb`
- `navis_base.ipynb`
- `Social_Network_Ads.csv`: dataset used for the classification experiments.

## Models Covered

- logistic regression
- k-nearest neighbors
- support vector machines
- decision trees
- random forests

## Getting Started

1. Clone the repository.
2. Open the notebooks in Jupyter.
3. Install common data science dependencies such as NumPy, pandas, matplotlib, and scikit-learn.
4. Run each notebook independently to compare the models.

## Notes

This repository is intended as a collection of classification exercises and baseline model implementations for study and comparison.
